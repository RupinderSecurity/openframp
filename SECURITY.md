# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in OpenFRAMP, please report it responsibly.

**Email:** rupinderpalsing@proton.me

Please do not open public GitHub issues for security vulnerabilities.

## Supported Versions

| Version | Supported |
| --- | --- |
| 1.x | Yes |

## Security Practices

- All dependencies are monitored via Dependabot
- Secret scanning and push protection are enabled
- Branch protection rules are enforced on main
